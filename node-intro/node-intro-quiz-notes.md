# node-intro-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What are the 3 components of a fullstack Web architecture?
  frontend: site/app that the user interacts with (presentation tier)
  backend: processing requests from frontend (logic tier)
  database: data storage that is used across all instances of frontend (data tier)
- What is Node.js?
  program that allows JS to be run outside of a browser
- What can Node.js be used for?
  commonly used for building backend, CL programs, automation
- What is a REPL?
  Read Evaluate Print Loop: interactive programming environment
- When was Node.js created?
  May 27, 2009
- What backend languages have you heard of?
  C++, C#, SQL, PHP
